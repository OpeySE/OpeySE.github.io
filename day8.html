Video tag

Introduction

Html5 uses a video tag to hold all the information about a video. Inside the video tag, you should add a control attribute. The controls attribute allows you to control the video e.g. volume. The video tag should also include the width and height attributes. If height and width are not set, the video size might not fit inside the browser window. I stored all my videos inside a folder, which I will be able to source from. The source element identifies what media resources are going to be displayed, this would include pictures and videos.
